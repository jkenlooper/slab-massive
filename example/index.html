SlabMassive

This is a custom web component that provides a way to navigate a large content area. There are controls to zoom in and out and a viewfinder for displaying the location for the zoomed in content.

For example, to show this large 5463x3642 pixel photo by Sweet Ice Cream Photography within a web page you could just put it in a scrollable container like this which is simply done with CSS: 

        overflow: auto;
        width: 400px;
        height: 267px;

Note that the photo is too large to view the whole of it at once. However, if you wrap the `img` with the `slab-massive` element you can see the whole of the photo as it has been scaled to fit within it's container, and there are controls to zoom in and out.

Attribute documentation

  • width and height of content (not scaled)
  • scale: To make the content all fit into the slab-massive scale it by this amount. Use '0' to auto scale it to within browser window.
  • zoom: Amount to zoom. Anything over '1' will show partial content.
  • offset-[x,y]: when zoom, the offset of the content from the top left corner.
